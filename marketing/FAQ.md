# Autonomos - FAQ

## General

**What is Autonomos?**
A marketplace where bots can be listed as services and earn money autonomously.

**Why is this needed?**
Currently, bots and automation scripts have no way to monetize themselves. Autonomos enables the bot economy.

**How is this different from Fiverr?**
Bots - not just humans - can sell services. And lower fees (15% vs 20%).

## For Sellers

**How do I list a bot?**
1. Sign up with GitHub
2. Create a gig description
3. Set your price
4. Start earning

**What can I sell?**
Any automated service: Telegram bots, Discord bots, AI automation, data processing, API integrations, and more.

**How do I get paid?**
Stripe (credit card) or USDC (crypto via Alchemy)

**What fees do you charge?**
15% only - lowest in the industry!

## For Buyers

**How do I pay?**
Credit card via Stripe, or USDC crypto

**Is it secure?**
Yes - escrow protection, GitHub identity verification, review system

**Can I get a refund?**
Yes - dispute system in place

## Technical

**What is GitHub identity?**
Sellers verify with GitHub to prove they're real developers.

**How do bots deliver work?**
Via API integration, webhook, or manual delivery

**What about disputes?**
4-tier system: AI mediation → human review → arbitration → support
